# 木樨映像默认视觉素材

本项目已使用 Codex 内置图片生成能力生成默认视觉素材，并复制到 `app/static/images/generated/`。后台“网站设置”可以上传新图片覆盖这些默认引用。

## 首页 Banner

- 用途：首页顶部大图
- 建议尺寸：1920x800
- 保存路径：`app/static/images/generated/home-banner.png`
- 代码默认引用：`site_settings.home_banner`
- 提示词：泸州高中木樨映像摄影社团首页 Banner，想象中的中国高中校园庭院，柔和晨光，木樨/桂花，相机，照片，胶片颗粒，清新文艺，摄影杂志封面感，无真实人物，无真实学校照片，无文字，无水印。

## 默认背景图

- 用途：登录页、注册页、后台初始化页背景
- 建议尺寸：1920x1080
- 保存路径：`app/static/images/generated/auth-background.png`
- 代码默认引用：`site_settings.auth_background`
- 提示词：安静校园走廊、柔和光影、桂花/木樨、胶片颗粒、浅色渐变、适合表单覆盖的留白，无真实人物，无真实学校照片，无文字，无水印。

## 用户预设头像

- 用途：用户资料页中的本地预设头像
- 保存路径：`app/static/images/avatars/avatar-01.webp` 至 `avatar-10.webp`
- 输出规格：512x512 WebP
- 通用提示词：清新摄影杂志风格的校园摄影社团成员头像，柔和自然光，干净单色背景，胸像居中，人物设定各不相同，无文字，无水印。
- 首批变化：短发眼镜与芥末色开衫、凌乱短发与藏蓝棒球外套、低马尾与浅色围巾、短卷发与森林绿针织背心、高马尾与珊瑚色外套、侧分短发与浅色牛仔、精灵短发与青绿色卫衣、波浪刘海眼镜与酒红毛衣、双辫与黄色雨衣、寸头与橄榄绿工装外套。
